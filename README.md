# معرض القدس للأجهزة المنزلية (Al Qods Store) 🏠✨

متجر إلكتروني متكامل للأجهزة المنزلية، مصمم بأحدث التقنيات ليوفر تجربة شراء سلسة وذكية.

## 🚀 المميزات
- **مساعد ذكي (محمد)**: بائع آلي يساعدك في اختيار المنتجات وإتمام الطلب باللهجة المصرية.
- **نظام تقييمات وتعليقات**: مراجعات حقيقية للمنتجات لزيادة الثقة.
- **لوحة تحكم كاملة**: للمسؤولين لإضافة وتعديل المنتجات وإدارة العروض.
- **نظام عروض ديناميكي**: عروض لفترة محدودة وخصومات تلقائية.
- **سلة تسوق ذكية**: تتبع المنتجات والأسعار بسهولة.
- **إتمام الطلب عبر واتساب**: ربط مباشر لتأكيد الطلبات.

## 🛠️ التقنيات المستخدمة
- **Frontend**: React.js, Vite, Tailwind CSS.
- **State Management**: Zustand.
- **Backend**: Node.js, Express (AI Chatbot Service).
- **Database**: Firebase (Firestore, Auth, Storage).
- **AI**: OpenRouter API.

## 📦 التشغيل المحلي
1. قم بتحميل المستودع: `git clone https://github.com/M-Max555/al-qods-store.git`
2. تثبيت المكتبات: `npm install`
3. تشغيل الموقع: `npm run dev`
4. تشغيل السيرفر: `cd server && npm install && npm start`

## 🌍 النشر
الموقع مهيأ للنشر على **Vercel** و **Render**.
